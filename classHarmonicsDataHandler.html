Detailed Description

Class for handling the result of a harmonics calculation.

This class is used to handle the result of a harmonics calculation. It provides functions to extract information from the raw harmonics data.

Parameters

Returns
Vector of pairs containing ell and Bn values.

This function returns the Bn data for a specific component along the magnet. The Bn data indicates the strength [T] of the component at certain points along the magnet (see ell). Only returns data within the set ell bounds.
