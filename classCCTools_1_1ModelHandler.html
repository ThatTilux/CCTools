Detailed Description

Class to manipulate a magnet model JSON file.

Provides functions to extract and manipulate drive values for custom cct harmonics.

Constructor & Destructor Documentation

CCTools::ModelHandler::ModelHandler ( const boost::filesystem::path &  json_file_path)

Construct a new ModelHandler object.

Parameters
json_file_pathPath to the JSON file.

This constructor copies the JSON file to a temporary folder. All changes inside this class are only made to the temporary file, the original file remains untouched. Path to the temporary JSON file can be accessed using getTempJsonPath().
