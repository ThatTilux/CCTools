enum HarmonicDriveParameterType
strong

Enum class representing harmonic drive parameter types for custom cct harmonics.

This enum class is used to represent the type of a harmonic drive parameter. For custom cct harmonics with an amplitude of constant, this value can only be constant. For custom cct harmonics with an amplitude of linear, this value can be offset or slope.
